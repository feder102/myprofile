Installing WordPress

Before beginning the installation and configuring of your new theme, you must first have WordPress already installed on a server. Here is a very detailed guide at Wordpress Codex that will walk you through every step.

Installing Theme

Uploading Through WordPress

To upload through your Dashboard, select “Appearance” and then “Themes”. Next, select the “Add New” button at the top of the page and then select the “Upload Theme” button. Choose Ryan theme .zip file and select “Install Now”. Once the theme is fully uploaded and installed, click “Activate” to activate the WordPress theme.

When uploading your theme with the installer, please ensure you are uploading the theme .zip file, not the entire package you downloaded. In this case, you will be uploading ryancv.zip.

FTP Upload

To manually upload your new WordPress theme, login with your credentials to your website and locate the wp-content folder in your WordPress install files. Upload the un-zipped Ryan WordPress Theme folder into the: wp-content/themes folder.
Once uploaded, activate the theme by heading to the “Themes” menu in the WordPress Dashboard.

Installing Plugins Required

Once you have installed and activated our theme you will be prompted to install required and recommended plugins for all theme features. Just click the link to Begin installing plugins.

Theme Options

All global settings can be found in ​Dashboard > Theme Options​. In addition, you can customize:

General

In "General" Tab you can edit: Sticky Menu Enabled, Hide vCard Panel on mobile.

vCard

In "vCard" Tab you can edit profile card: Background Image, Photo, Title, Subtitle, Social Links, Buttons.

Styling

In "Styling" Tab you can edit: Background Type(Image, Video, Gradient), Section Animations, Theme Color, Heading Title font / color / size, Paragraph font / color / size, Bottom Line color.

Blog Posts

In "Blog Posts" Tab you can disable/enabled: categories, blog subtitle, edit subtitle, disable/enabled post excerpt, show/hide featured image.

Portfolio

In "Portfolio" Tab you can disable/enabled single portfolio page.

Sidebar

In "Sidebar" Tab you can disable/enabled sidebar.

Google Maps

In "Google Maps" Tab you can edit API Key.

Home Page

To create Home Page, please do the following:
1. Go to Dashboard > Pages > Add New.
2. Enter your Page Title "Home".
3. In the tab "Page Attributes" > "Template", select "Layout builder".
4. In the tab "Layout Builder" click to "Add Module" button and select modules.
5. In the appeared new module tab fill the fields.
6. In the tab "Publish" click to "Publish" button.
7. Go to Dashboard > Settings > Reading. In the field "Your homepage displays" > "A static page (select below)" > "Homepage:" select "Home"
8. Click to "Save Changes" button.

Resume Page

To create Resume Page, please do the following:
1. Go to Dashboard > Pages > Add New.
2. Enter your Page Title "Resume".
3. In the tab "Page Attributes" > "Template", select "Layout builder".
4. In the tab "Layout Builder" click to "Add Module" button and select modules.
5. In the appeared new module tab fill the fields.
6. In the tab "Publish" click to "Publish" button.

Works Page

To create Portfolio Page, please do the following:
1. First your need to create Portfolio categories, go to Portfolio > Portfolio categories, enter category name in fields Name & Slug. Click to "Save Changes" button.
2. Go to Portfolio > Add Portfolio.
3. Enter your Work Name, Description, button url link, category and the tab Featured Image upload & select preview image. In the tab "Publish" click to "Publish" button.
4. Go to Dashboard > Pages > Add New.
5. Enter your Page Title "Works".
6. In the tab "Page Attributes" > "Template", select "Layout builder".
7. In the tab "Layout Builder" click to "Add Module" button and select "Portfolio".
8. In the appeared new module tab fill the fields "Title". The field "Portfolio items" click to "Add Item" button and select your works.
9. In the tab "Publish" click to "Publish" button.

Contacts Page

To create Contacts Page, please do the following:
1. Go to Dashboard > Contact > Contact Forms > Contact Form 1.
2. In the tab "Form", enter this code lines:

<div class="row">
   <div class="col col-d-6 col-t-6 col-m-12">
      <div class="group-val">
         [text* your-name placeholder "Full Name"]
      </div>
   </div>
   <div class="col col-d-6 col-t-6 col-m-12">
      <div class="group-val">
         [email* your-email placeholder "Email Address"]
      </div>
   </div>
   <div class="col col-d-12 col-t-12 col-m-12">
      <div class="group-val">
         [textarea* your-message placeholder "Your Message"]
      </div>
   </div>
</div>
<div class="align-left">
<button type="submit" class="button"><span class="text">Send Message</span><span class="arrow"></span></button>
</div>

3. In the tab "Mail", enter this your email address.
4. Go to Dashboard > Pages > Add New.
5. Enter your Page Title "Contacts".
6. In the tab "Page Attributes" > "Template", select "Layout builder".
7. In the tab "Layout Builder" click to "Add Module" button and select "Contacts".
8. In the appeared new module tab fill the fields.
9. In the tab "Layout Builder" click to "Add Module" button and select "Contacts Form".
10. In the appeared new module tab fill the fields.
11. In the tab "Publish" click to "Publish" button.

Blog Pages

To create Blog Pages, please do the following:
1. Go to Dashboard > Pages > Add New.
2. Enter your Page Title "Blog".
3. In the tab "Page Attributes" > "Template", select "Layout builder".
4. In the tab "Layout Builder" click to "Add Module" button and select "Blog".
5. In the tab "Publish" click to "Publish" button.
6. Go to Dashboard > Settings > Reading.
In the field "Your homepage displays" > "A static page (select below)" > "Posts page:" select "Blog"
7. Click to "Save Changes" button.
8. Go to Dashboard > Posts > Add New.
9. Enter your Post Title, Description, choose category and the tab Featured Image upload & select preview image. In the tab "Publish" click to "Publish" button.

Create Menu

To create Create/Edit Menu, please do the following:
1. Go to Dashboard > Appearance > Menus.
2. Select create a new menu at the top of the page.
3. Enter a name for your new menu in the Menu Name box.
4. Click the Create Menu button.
5. Select the Pages that you want to add by clicking the checkbox next to each Page's title. Click the Add to Menu button. 6. If you want Menu with icons, go to Appearance > Menus > Menu Structure > Menu item and select icon.
